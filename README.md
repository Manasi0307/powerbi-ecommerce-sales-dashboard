The project analyzes sales performance from different perspectives to derive meaningful business insights.

Project Overview:
This project focuses on understanding sales, profit, quantity, and shipping cost trends across categories, markets, regions, and shipping modes using interactive Power BI visuals.

Key Insights:
 Total Sales: 12.64M
 Total Profit: 1.47M
 Total Quantity Sold: 178K
 Total Shipping Cost: 1.35M

Analysis Covered:
-Sales by Category, Market, Country, State, and Ship Mode.
-KPIs using cards.
-Interactive slicers for Category and Market.

Tools used- Power BI, interactive insight generation.

This dashboard helps quickly identify top-performing categories, markets, and efficient shipping modes for better decision making.
